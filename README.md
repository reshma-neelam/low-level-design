# low-level-design
practice repo for LLD topics

Game is implemented to support N board size with N-1 players.
By default, number of players is two -
One human and One Bot

Below Winning strategies are implemented and are pluggable:
Row, Column, Diagonals

More winning strategies can be plugged.
By default, all 3 are added.

One Easy Bot playing level is implemented.

Implementation for Medium and Hard playing levels by BOTs can be added
